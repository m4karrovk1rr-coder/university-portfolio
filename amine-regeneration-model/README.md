# Разработка программного комплекса для моделирования аминной регенерационной колонны

**Дисциплина:** Программирование и алгоритмизация

### Суть проекта:
Комплексный инженерно-аналитический проект по созданию статической и динамической имитационной модели сложного технологического объекта — аминной регенерационной колонны (десорбера), используемой в системах очистки природного газа. Проект демонстрирует навыки системного декомпозирования сложных физико-химических процессов в программный алгоритм.

### Решенные задачи:
* **Системный анализ объекта:** Описание технологической схемы и декомпозиция процесса на материальные и тепловые потоки.
* **Алгоритмизация:** Разработка математических алгоритмов для расчета профилей концентраций кислых газов ($H_2S$, $CO_2$) по высоте колонны.
* **Имитационное моделирование:** Оценка динамических характеристик процесса, расчет мощности ребойлера регенератора и температуры выходных потоков.
* **Динамический анализ:** Моделирование поведения системы (переходных процессов) при изменении внешних возмущений и управляющих воздействий.

### Технологии и инструменты:
* **Python** (разработка основного расчетного модуля, моделирование функции `model_amine_absorber()`, профилирование концентраций компонентов).
* **MATLAB / Simulink** (построение структурных схем и исследование динамических режимов работы колонны).

### Структура папки:
* `report.docx` — подробная пояснительная записка с описанием технологического процесса, системы уравнений, блок-схем алгоритмов и анализом результатов.
* `amine_model.py` — программный модуль на Python, реализующий расчет статического режима, материальных балансов и профилей концентраций.
