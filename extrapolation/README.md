# Прогнозная экстраполяция на основе линеаризованных трендов

**Дисциплина:** Теория и методы прогнозирования

### Суть проекта:
Научно-исследовательская работа, направленная на изучение методов анализа временных рядов и построение прогнозных моделей. Основной упор сделан на метод линеаризации нелинейных трендов, который позволяет сводить сложные зависимости к линейному виду для повышения точности и стабильности долгосрочных прогнозов технологических параметров.

### Решенные задачи:
* **Анализ временных рядов:** Выявление основных тенденций (трендов) в массивах числовых данных.
* **Линеаризация нелинейных зависимостей:** Применение математических преобразований для приведения нелинейных моделей к линейному виду.
* **Статистический анализ качества:** Расчет остаточной дисперсии, коэффициентов детерминации ($R^2$) и средних ошибок аппроксимации.
* **Экстраполяция:** Построение точечных и интервальных краткосрочных/долгосрочных прогнозов.

### Инструменты:
* **Mathcad** (проведение сложных математических расчетов, линеаризация, решение систем уравнений).
* **Microsoft Excel** (табличное моделирование, визуализация трендов и прогнозных коридоров).

### Структура папки:
* `report.pdf` — полный аналитический отчет со всеми математическими выкладками, оценками погрешностей и прогнозными графиками.
