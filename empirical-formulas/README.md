# Построение эмпирических формул методами аппроксимации

**Дисциплина:** Введение в информационные технологии / Информатика

### Суть проекта:
Проект посвящен статистической обработке экспериментальных данных для установления математической зависимости между давлением взрыва сульфидной пыли и концентрацией различных наполнителей. В ходе работы проведено исследование различных видов аппроксимации (линейной, квадратичной, экспоненциальной) для нахождения наиболее точной математической модели.

### Решенные задачи (в фокусе Системного анализа):
* **Анализ данных:** Сглаживание исходных массивов данных и фильтрация случайных отклонений.
* **Математическое моделирование:** Расчет параметров эмпирических формул методом наименьших квадратов (МНК).
* **Верификация моделей:** Оценка точности и выбор оптимального уравнения (в данном кейсе — экспоненциальная аппроксимация).
* **Контрольный расчет и визуализация:** Написание скрипта для графического сравнения моделей и верификации расчетов, сделанных в Excel и Mathcad.

### Стек технологий и инструментов:
* **Python** (библиотека `matplotlib` для визуализации графиков аппроксимации).
* **Microsoft Excel / Mathcad** (первичный статистический анализ и расчет МНК).

### Структура папки:
* `report.docx` — текст курсовой работы (пояснительная записка) со всеми формулами, графиками и выводами.
* `approximation.py` — скрипт на Python для построения графиков линейной, квадратичной и экспоненциальной аппроксимации.
